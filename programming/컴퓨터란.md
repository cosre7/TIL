# 컴퓨터란?
### 컴퓨터의 구성
- 중앙처리장치(CPU)
    - 산술/논리 연산    
        ex) 3+4 연산
    - 흐름 제어   
        ex) 3+4 연산을 반복적으로 하게 해주는 것, 무한 OR n번   

- 주기억장치(Memory)   
    - 임시 저장소
- 다양한 입/출장치(HDD/ODD/..)
    - 데이터 소스/영구 저장소
        - 영구저장소 :  주기억 장치에서 저장되는 것은 임시, 영구적으로 저장하기 위해 필요
        - 콘솔(키보드, 마우스) : 사용자가 기계(컴퓨터)를 사용하기 위한 조종장치
```
입출력 장치의 조합에 따라서 컴퓨터의 종류/용도가 달라진다.
```
