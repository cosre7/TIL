# 컴퓨터 프로그래밍이란?
- 프로그래밍이란 수행 절차를 기록하는 것   
- 컴퓨터 프로그램이란 컴퓨터를 이용한 절차를 만들어 놓은 것   
```
1. 시작 : 시작이 한번만 시행되는 것이 아니라 CU를 통해 반복적으로 수행되게 된다. -> 일반적인 프로그램 형태
2. 읽어오기 : 보조장치에서 메모리를 읽어온다.
3. 메뉴 출력 ex) 모니터
4. 형태에 따라 
  - 분기 -> CU -> 절차를 만들어내는 능력을 가짐 => 가장 중요한 것
  - 연산 -> ALU
5. 종료
```
---   
- 상황에 따라 비교되는 용어   
재료 <------> 수행절차   
플랫폼 <------> 프로그램   
하드웨어 <------> 소프트웨어
